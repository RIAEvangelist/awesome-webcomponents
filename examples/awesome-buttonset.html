The awesome-buttonset element will trigger a change event when a button is clicked. Check the e.target.value to see which button index is currently active.

Paste the following into your console to see an example. 


            
                document.querySelector('awesome-buttonset:first-of-type').addEventListener(
                    'change',
                    buttonsetChange
                );

                document.querySelector('awesome-buttonset:last-of-type').addEventListener(
                    'change',
                    buttonsetChange
                );

                function buttonsetChange(e){
                    console.log(e.target);
                    console.log(`The value is : ${e.target.value}`);
                }
